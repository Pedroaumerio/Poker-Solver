# ♠️ Poker Solver Study

Treine suas decisões no poker como um profissional utilizando conceitos de **GTO (Game Theory Optimal)** e análise de **EV+**.

---

## 🚀 Sobre o projeto

O **Poker Solver Study** é uma aplicação web criada para ajudar jogadores a melhorarem sua tomada de decisão no poker através de simulações, ranges e estudos estratégicos.

A plataforma é focada em três principais modalidades:

* 🏆 **Torneios (MTT)**
  Estratégias com foco em ICM, pressão de blinds e stacks curtos.

* 🎰 **Spin & Go**
  Jogo 3-handed com decisões rápidas e ranges amplos.

* 💰 **Cash Game**
  Estratégias deep stack com foco em exploração de leaks e maximização de valor.

---

## 🧠 Funcionalidades

* 📊 Estudo de ranges (GTO)
* ♾️ Simulação de mãos
* 📈 Análise de decisões EV+
* 🎯 Treinamento prático por cenário
* 🧪 Simulador de situações personalizadas
